Bulk File Renamer & Smart Organizer:
  A Python Tkinter desktop app for batch renaming and organizing files inside a selected folder. It supports drag-and-drop folder selection, browse-based selection, live previews, undo for the last action, and a log viewer.

Features:
- Batch rename files with prefix, suffix, find/replace, and auto numbering.

- Organize files into subfolders by type, extension, modified date, or file size.

- Select folders by browse button or drag and drop.

- Preview changes before applying them.

- Undo the last rename or move batch.

- View action history from the built-in log tab.

Requirements:
- Python 3.x
- tkinter
- tkinterdnd2

Install:
- pip install tkinter
- pip install tkinterdnd2

Run:
- python main.py

Notes:
- Drag and drop support depends on tkinterdnd2.

- The app creates a CSV log file in the selected folder.
